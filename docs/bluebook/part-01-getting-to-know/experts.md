# 专家

🧠 「专家」是更专业的朋友。普通助理像通才，专家像专科医生——专攻一类问题。

<div class="tag-row">
  <span class="tag">难度：进阶</span>
  <span class="tag">用时：12 分钟</span>
  <span class="tag">WorkBuddy v5.2.5</span>
</div>

## 你将拿到什么

- 一份「专家使用清单」：什么时候找专家、怎么挑、怎么问。
- 知道专家不是神，是更聚焦的提示词。

## 什么时候用

- 写完方案想让人帮忙挑刺。
- 多个方案拿不定主意。
- 涉及专业领域：法律、营销、财务、产品。
- 想做对抗测试——找专家专门反驳自己。

## 专家 vs 普通助理

- **助理**：通用、能干很多事。
- **专家**：聚焦一类问题，会主动用该领域的标准、量表、常见坑。

## 三步搞定

1. 侧栏点「专家」，选择合适的方向（产品/营销/法律/财务等）。
2. 第一轮把背景、约束、目标说清楚。
3. 让专家先用一句话给你"判断 + 理由 + 风险"，再展开。

## 提示词：让专家挑刺

```text
角色：你是[产品/营销/法务]专家。

背景：____________________

我已经有的方案：____________________

请先回答三个问题：
1. 这个方案最大的风险是什么？
2. 哪一步最可能走偏？
3. 如果只能改一处，应该改什么？

不要寒暄，不要给"做得好"的夸奖。
```

## 提示词：让专家对比两个方案

```text
请用以下结构对比方案 A 和方案 B：
- 目标达成度（满分 5 分）
- 实施成本（满分 5 分，5 = 最低成本）
- 主要风险
- 我的建议
- 建议补充的事实信息

最后给一句话总结："如果只能选一个，我选 ____，因为 ____"。
```

## 提示词：对抗测试

```text
我要你扮演"最挑剔的反对者"。

我的方案：____________________

请列出 5 个反对理由，按风险从高到低。
每个反对理由给一个我需要补的事实。
```

## 常见失败和修复

- ❌ 把专家当万能助理 → ✅ 专家擅长本领域，超出范围让它主动说"这个我判断不了"。
- ❌ 专家给的建议不带证据 → ✅ 让它"先给判断 + 理由 + 风险"，再展开。
- ❌ 看完建议就照做 → ✅ 一定自己复核：它引用的标准/数据来源是否最新。

## 练一练

- 找一个写完的方案，用"挑刺"提示词让专家反驳。
- 找两个备选方案让专家对比。
- 用对抗测试列出 5 个反对理由。
