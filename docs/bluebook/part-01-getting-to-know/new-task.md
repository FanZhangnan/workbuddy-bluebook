# 新建任务

✨ 「新建任务」是你和 WorkBuddy 聊天最轻的入口——单次问答、临时改写、快速总结都用它。

<div class="tag-row">
  <span class="tag">难度：入门</span>
  <span class="tag">用时：8 分钟</span>
  <span class="tag">WorkBuddy v5.2.5</span>
</div>

## 你将拿到什么

- 一套「角色 + 任务 + 约束 + 输出」提示词模板。
- 3 个常见场景的完整提示词示例。

## 什么时候用

- 临时问个问题：把一段话改得更口语。
- 轻量改写：把正式语气改成朋友聊天的语气。
- 快速总结：把一段长文压成 3 个要点。

## 区别于「助理」

「新建任务」是**一次性对话**，关掉就结束；「助理」会持续记得你说的话。这章只看「新建任务」。

## 三步搞定

1. 在左侧栏点「新建任务」。
2. 把要处理的内容贴进输入框，按模板写提示词。
3. 看回结果，满意就复制走；不满意就在原任务里追问。

## 提示词模板

```text
角色：你是[职业/角色]，擅长[能力]。

任务：把下面这段内容[动作：改写/总结/翻译/扩写]。

约束：
- 长度：X 字左右
- 语气：[正式/口语/活泼]
- 不要：[不要做的事]

输出格式：
- 标题
- 3 个要点
- 1 句结论

输入内容：
[在这里粘贴]
```

## 三个常用示例

### 例 1：把会议纪要改写成周报

```text
角色：你是产品经理助理。

任务：把下面会议纪要改写成给老板的周报。

约束：
- 长度：150 字内
- 语气：正式但不官腔
- 不要：原话照抄，要写人话

输出格式：
- 本周完成
- 下周计划
- 需要老板决定的事

输入内容：
[粘贴会议纪要]
```

### 例 2：把客户邮件改得更礼貌

```text
角色：你是客户成功经理。

任务：把下面这段话改得更礼貌、更专业，但保持原意。

约束：
- 不要改变事实
- 加一句致谢
- 长度不变

输入内容：
[粘贴原文]
```

### 例 3：把长文压成 3 个要点

```text
角色：你是研究助理。

任务：把下面这篇长文压成 3 个要点 + 1 句金句。

输入内容：
[粘贴长文]
```

## 预期效果

- 你会得到一段可以直接用的成稿。
- 知道什么时候加约束（长度、语气、不能做什么）。

## 常见失败和修复

- ❌ 输出太啰嗦 → ✅ 提示词里写「长度：X 字内」并先给示例。
- ❌ 改了原意 → ✅ 写「不要改变事实」并提供原文关键句。
- ❌ 风格不对 → ✅ 给一段「风格示例」让 WorkBuddy 对齐。

## 练一练

- 找一段你工作里写过的草稿，用模板改成正式/口语两个版本。
- 改写一封邮件，加「不要改变事实」的约束。
- 总结一篇公众号长文为 3 个要点。

## 作品集节点

把 3 段提示词模板存到「我的文件 → 作品集 → 提示词库」。

## 版本说明

适用于 WorkBuddy v5.2.5。
