# 腾讯文档 / ima / 乐享

📚 WorkBuddy 可以直接读你授权的「腾讯文档 / ima 知识库 / 乐享知识库」——不用先导出再贴。

<div class="tag-row">
  <span class="tag">难度：入门</span>
  <span class="tag">用时：10 分钟</span>
  <span class="tag">WorkBuddy v5.2.5</span>
</div>

## 你将拿到什么

- 一份「知识库使用清单」：什么时候用哪个、怎么授权、怎么提问。
- 知道三个知识库的差别和典型场景。

## 三个知识库差别

| 知识库 | 适合放什么 | 典型场景 |
|---|---|---|
| **腾讯文档** | 个人/团队协作文档 | 会议纪要、项目计划、共享表格 |
| **ima 知识库** | 个人学习/资料整理 | 公众号收藏、读书笔记、素材库 |
| **乐享知识库** | 企业内训/制度文档 | 公司制度、产品手册、培训材料 |

## 什么时候用

- 让 WorkBuddy 读你腾讯文档里的会议纪要，整理成行动项。
- 让它搜 ima 知识库里你收藏过的公众号文章，回答相关问题。
- 让它读乐享里的产品手册，帮你写客户答疑。

## 三步搞定

1. 侧栏点「更多 → 腾讯文档 / ima 知识库 / 乐享知识库」。
2. 选择你要授权的库（最小范围）。
3. 在提示词里说"请读 XX 知识库中的 YY 文档"。

## 提示词：用知识库回答

```text
请读 [腾讯文档/ima/乐享] 中的 [文档名或目录]，
回答：____________________

回答要求：
- 引用时告诉我"来自 XX 文档第 N 段"；
- 如果文档里没答案，明确说"未在文档中找到"；
- 不要凭印象编。
```

## 提示词：跨知识库汇总

```text
请同时读：
- 腾讯文档：[会议纪要目录]
- ima 知识库：[我收藏的素材库]

任务：找出 [主题] 相关的所有信息，按时间倒序列出来。

注意：来自不同来源的信息要分开标注。
```

## 常见失败和修复

- ❌ 授权范围太大 → ✅ 选具体文档或目录，不要授权整个工作区。
- ❌ 引用错位 → ✅ 提示词里加"按段落引用"。
- ❌ 知识库里没答案但 WorkBuddy 编了 → ✅ 提示词里加"未找到就明说"。

## 练一练

- 找一个腾讯文档，让 WorkBuddy 读并回答 3 个问题。
- 让它读 ima 知识库的某个收藏夹，整理成提纲。
- 写一份"我授权了哪些知识库 + 范围"清单。

## 作品集节点

存"知识库授权清单"到「作品集 → 授权清单」。
