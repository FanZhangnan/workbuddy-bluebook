# 我的文件

📁 「我的文件」是 WorkBuddy 帮你保管资料的地方。上传、整理、检索、分享——都在这里。

<div class="tag-row">
  <span class="tag">难度：入门</span>
  <span class="tag">用时：8 分钟</span>
  <span class="tag">WorkBuddy v5.2.5</span>
</div>

## 你将拿到什么

- 一份「文件夹结构模板」。
- 知道哪些文件适合上传、哪些不要。

## 什么时候用

- 让 WorkBuddy 读你的资料回答问题。
- 把常用素材/模板集中管理。
- 做内容时让 WorkBuddy 引用你授权的文件。

## 哪些不要传

- 包含密码、API Key、Token 的文件。
- 客户/同事的隐私数据（未脱敏）。
- 受版权保护且未授权复制的文件。

## 三步搞定

1. 侧栏点「更多 → 我的文件」。
2. 建一个**稳定结构**——别每次新建。
3. 上传文件时，提示词里说明"请用我的文件中的 XX.md 作为参考"。

## 推荐文件夹结构

```text
我的文件/
├── 00-作品集/
│   ├── 01-起步
│   ├── 02-项目
│   └── 03-提示词库
├── 10-工作/
│   ├── 当前项目
│   ├── 模板
│   └── 资料
├── 20-学习/
└── 30-私事/
```

编号前缀让文件按你想要的方式排序，**不要在名字里加日期**——日期会让检索变难。

## 提示词：让 WorkBuddy 用我的文件

```text
请阅读我「我的文件/10-工作/当前项目/背景.md」的内容，
然后回答：____________________

引用时告诉我"来自 背景.md 第 X 段"。
```

## 提示词：整理文件

```text
请帮我整理「我的文件/10-工作/当前项目」：
1. 按主题分到 3–5 个子文件夹；
2. 重命名为"主题_简短描述.md"格式；
3. 给我一个目录索引（每个文件一句话说明）。

不要移动我标了"重要"的文件。
```

## 常见失败和修复

- ❌ 文件太多找不到 → ✅ 定期整理；用编号前缀分组。
- ❌ 上传了含密钥的文件 → ✅ 上传前 grep 一下"password/secret/api_key"。
- ❌ 分享给同事时不知道权限 → ✅ 用最小权限；链接加过期时间。

## 练一练

- 建上面那个文件夹结构。
- 上传 3 份工作文件，提示词里让 WorkBuddy 引用回答。
- 写一份"我哪些资料不能传"清单。

## 作品集节点

把"文件夹结构 + 不能传清单"存到「作品集 → 文件清单」。

## 版本说明

适用于 WorkBuddy v5.2.5。文件存储与权限以官方说明为准。
