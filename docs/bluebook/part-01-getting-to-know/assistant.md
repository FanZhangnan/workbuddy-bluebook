# 助理

💬 「助理」是 WorkBuddy 的常驻聊天对象。和「新建任务」不同，**它会记得你说过的内容**。

<div class="tag-row">
  <span class="tag">难度：入门</span>
  <span class="tag">用时：10 分钟</span>
  <span class="tag">WorkBuddy v5.2.5</span>
</div>

## 你将拿到什么

- 一份「助理使用清单」——什么时候用，怎么用，怎么收尾。
- 3 个长对话的实用技巧。

## 什么时候用

- 一件事要来回聊好几次：写方案、改稿子、迭代想法。
- 需要 WorkBuddy 记住前面定下的目标、风格、约束。
- 想要一个**持续陪伴**的角色，而不是一次性问答。

## 助理 vs 新建任务

| | 新建任务 | 助理 |
|---|---|---|
| 上下文 | 不保留 | 保留 |
| 适合 | 单次问答 | 持续协作 |
| 结束 | 关掉就走 | 你说"结束"或新开会话 |

## 三步搞定

1. 在侧栏点「助理」，开一个会话，给它起个名字。
2. **第一轮就把目标、风格、约束说清楚**——别让它猜。
3. 之后每轮只说"还要改什么"，它会记得前面所有内容。

## 提示词：开启一段助理对话

```text
你好，我们要做一件事：____________________

风格要求：
- 语气：____________________
- 长度：____________________
- 受众：____________________

每次给我输出前，先告诉我你打算怎么做（3 句话以内）。
我回复「继续」后再给完整答案。
```

## 三个实用技巧

### 1. 每轮都「追问」而不是「重启」

错误：发现答案不对，开新会话再说一遍。
正确：在当前会话里说"刚才第三点不对，应该是 X，重新写"。

### 2. 隔一段时间做「小结」

每隔 5–10 轮说一次"现在我们定下来的是：……"，让 WorkBuddy 帮你校准。

### 3. 收尾时让它给你一份「成稿」

在长对话的最后说："请把上面所有内容整理成一份可以直接用的成稿，列出来源和需要我确认的点。"

## 常见失败和修复

- ❌ 助理"忘了"前面说的话 → ✅ 检查是不是开了新会话；同一个助理内会保留。
- ❌ 输出越来越跑偏 → ✅ 每 5 轮做一次小结，并重申约束。
- ❌ 想要导出整段对话 → ✅ 看对话区右上角的「导出」按钮（不同版本可能略有差异）。

## 练一练

- 开一个助理，让它帮你打磨一段自我介绍。每轮只追问一个改动。
- 让它帮你写一份活动方案，每 5 轮小结一次。
- 收尾时让助理给你一份成稿。

## 作品集节点

把这次助理对话的关键提示词和成稿存到「作品集 → 提示词库」。

## 版本说明

适用于 WorkBuddy v5.2.5。不同版本"助理"功能位置可能略有差异。
