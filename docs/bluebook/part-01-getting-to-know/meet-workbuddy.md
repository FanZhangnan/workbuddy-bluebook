# 认识 WorkBuddy 整体

👋 第一次打开 WorkBuddy 容易一脸懵——按钮太多。这章用 5 分钟让你对主界面心里有数。

<div class="tag-row">
  <span class="tag">难度：入门</span>
  <span class="tag">用时：10 分钟</span>
  <span class="tag">WorkBuddy v5.2.5</span>
</div>

## 你将拿到什么

- 一张「界面地图」——以后看任何按钮都能定位。
- 一份「我常用什么」清单——知道自己更依赖哪几个功能。

## 什么时候用

- 第一次打开 WorkBuddy。
- 介绍给同事/朋友时自己梳理思路。
- 想换模式、改设置时回来对照。

## 界面长什么样

主界面大致分三块：

- **左侧侧栏**：从上到下依次是「新建任务」「助理」「项目」「专家 · 技能 · 连接器」「自动化」「更多」。点击「更多」会展开「我的文件、腾讯文档、ima 知识库、乐享知识库、灵感」等子菜单。
- **主对话区**：你打字、WorkBuddy 回话的地方。
- **顶栏/设置区**：通常包含模式切换、主题、搜索、版本号。

把 WorkBuddy 想象成一个**坐在你左边的同事**——左侧栏是这位同事的工具箱，主对话区是你们正在聊的事。

## 三步搞定

1. **打开 WorkBuddy v5.2.5**。看左侧栏，先点几个按钮感受位置，不需要真做事。
2. **点开「更多」**，看一眼展开的子菜单。这就是你以后找资料、接知识库的地方。
3. **顶栏找到模式切换**（Work / Code / Design / Plan 等），默认应该是 Work 模式。

## 提示词：让 WorkBuddy 帮你介绍自己

```text
我刚打开 WorkBuddy v5.2.5，不太认识界面。
请用三段话告诉我：
1. 左侧栏每个按钮分别是干什么的；
2. 主对话区我可以输入什么；
3. 我应该先点哪个按钮开始第一个任务。
不要说术语，按钮用「左侧从上到下第 N 个」描述。
```

## 预期效果

- 你能在脑海里画出一张「界面地图」。
- 知道新建任务、助理、项目是三个最常用的入口。
- 知道「更多」藏着资料库和灵感。

## 常见失败和修复

- ❌ 找不到模式切换 → ✅ 看顶栏右侧或左下角的小图标，hover 一下会显示模式名。
- ❌ 左侧栏显示不全 → ✅ 鼠标移到左侧边线，会出现拖动条。
- ❌ 不知道现在用哪个模式 → ✅ 看主对话区顶部的小徽标，WorkBuddy 会显示当前模式。

## 练一练

- 打开 WorkBuddy，把侧栏每个按钮点一遍，记下自己的第一感受。
- 用上面的提示词问 WorkBuddy 自己，对比它说的和你看到的界面。
- 列出 3 个你接下来最想学的功能。
