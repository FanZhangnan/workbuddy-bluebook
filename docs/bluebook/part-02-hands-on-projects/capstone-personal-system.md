# 收官·把方法变成你的工作系统

🏁 学完前面 18 章——是时候把"我用过的好东西"沉淀成你专属的工作系统。

<div class="tag-row">
  <span class="tag">难度：综合</span>
  <span class="tag">用时：30 分钟</span>
  <span class="tag">WorkBuddy v5.2.5</span>
  <span class="tag rose">收官</span>
</div>

## 你将拿到什么

- 1 份**个人工作系统清单**：技能/连接器/自动化/助理模板。
- 1 份**季度复盘模板**。

## 什么时候用

- 学完所有项目后做总复盘。
- 季度初规划工作时。
- 觉得 WorkBuddy "用得太散"想集中管理时。

## 跟着做

### 阶段 1：盘点"我用过的"（10 分钟）

打开**助理**。

```text
请帮我盘点过去 [时间段] 我用过的：
1. 模式（Work / Plan / Design / Code 哪个用得多）
2. 技能（哪些技能我装了且用过）
3. 连接器（我接了哪些）
4. 自动化（我建了哪些）
5. 助理/专家（常用的有谁）
6. 项目（我做过的项目）

请按"用得多/用得少/没用过"分三档。
```

### 阶段 2：保留"高价值的"（5 分钟）

```text
基于上面盘点：
1. 哪些是"用得多 + 高价值"，必须保留
2. 哪些是"用得少 + 高价值"，需要复盘为什么少
3. 哪些是"没用过 + 低价值"，建议清理

给我一份"保留/复盘/清理"三栏清单。
```

### 阶段 3：建"工作流模板"（10 分钟）

把高频流程做成可复用模板：

- **周报流**：触发（每周五）→ 助理（周报提示词）→ 文件（周报文件夹）。
- **客户答疑流**：触发（收到客户邮件）→ 助理（用知识库答）→ 发邮件。
- **品牌站内容流**：触发（每周三）→ 助理（选题）→ 助理（写稿）→ 助理（配图建议）→ 发布。

```text
请把上面 [X 个流] 各写一份"提示词 + 自动化"组合模板。

格式：
- 名称
- 触发
- 步骤
- 提示词
- 失败处理
```

### 阶段 4：建个人专家（可选，5 分钟）

如果你有反复需要的判断/复盘工作，可以建一个个人专家。

```text
请帮我设计一个 [领域] 个人专家：
- 专长：[3 条]
- 必问问题：[5 条]
- 拒绝回答什么：[3 条]
- 引用来源：要求 [____]
```

### 阶段 5：季度复盘模板（5 分钟）

```text
请给我一份季度复盘模板：

- 本季完成的项目（PB-01..PB-05 状态）
- 跑起来的自动化（哪些稳定，哪些要修）
- 新学的技能（用了哪些）
- 没解决的事（带进下季）
- 下季 3 个重点
```

## 验收清单

- [ ] 盘点三档（用得多/少/没用）。
- [ ] 清理了 30% 以上没用的技能/连接器。
- [ ] 至少 1 个"工作流模板"被文档化。
- [ ] 季度复盘模板建好。
- [ ] 给品牌站 PB-06 留了入口（第 16 章旗舰项目的延续）。

## 常见失败和修复

- ❌ 工具越多越乱 → ✅ 定期清理，保留"高价值 + 高频"。
- ❌ 模板写了不用 → ✅ 第一次跑就跑真实任务，别只是"准备"。
- ❌ 没复盘 → ✅ 固定季度复盘日（如每季度首周一上午）。

## 练一练

- 跑完整 5 阶段，得到一份"个人工作系统"清单。
- 把清单 + 模板 + 复盘存到「作品集 → 04-工作系统」。

## 作品集节点

存"个人工作系统 + 复盘"到「作品集 → 04-工作系统」。
