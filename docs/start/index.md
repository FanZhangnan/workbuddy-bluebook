# 开始阅读

这套蓝皮书可以**两小时上手**——你只要准备一个真实小任务，按章节做下去就行。

## 开始前的三条提醒

1. 不要上传密码、身份证件、未授权的客户资料和人事敏感数据。
2. 界面、可用技能和连接器会随版本变化，本书内容以**当前官方版本**为准。
3. 所有发布、发送、覆盖、删除和跨系统写入动作，**由你做最后确认**。

## 推荐顺序

| 阶段 | 阅读 | 目标 |
|---|---|---|
| 认识工具 | 第 1–5 章 | 看懂主界面，5 个最常用入口（新建任务/助理/项目/模式/更多） |
| 进阶能力 | 第 6–9 章 | 学会用专家、技能、连接器、自动化 |
| 资料管理 | 第 10–12 章 | 管好我的文件、知识库和灵感 |
| 项目实战 | 第 13–18 章 | 从 5 分钟汇报到品牌网站，做 6 个真实项目 |
| 收官 | 第 19 章 | 把方法沉淀成你的工作系统 |

## 不知道从哪里开始？

- **有明确问题**：去[帮你解决](/solve/)。
- **想系统学**：从[第 1 章](/bluebook/part-01-getting-to-know/meet-workbuddy)开始。
- **想直接做项目**：从[项目一·5 分钟汇报](/bluebook/part-02-hands-on-projects/project-report)开始。
- **想看完整路线**：[学习路线](/paths/)。
