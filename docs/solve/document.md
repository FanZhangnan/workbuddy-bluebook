# 写或修改文档

## 快速安全方案

先写任务合同，再依次完成提纲、初稿、评审和修订。每一阶段只做一种判断：提纲看结构，初稿看完整，评审查证据与风险，修订只接受明确意见。保留修订记录，不让“润色”悄悄改变事实。
