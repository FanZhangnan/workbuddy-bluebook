# 研究与总结

## 快速安全方案

1. 把问题缩成一个决策句，写清时间范围和排除项。
2. 建立来源台账：标题、发布者、日期、链接、原文摘录、可信度。
3. 要求输出区分“来源事实、综合推断、不确定项、待验证问题”。
4. 人工打开关键来源，核对数字、日期和上下文。
