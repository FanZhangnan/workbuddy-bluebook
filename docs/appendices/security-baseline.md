# 安全基线

## 输入前

- 按公开、内部、敏感、禁止处理分类；默认最小化。
- 删除密钥、身份证件、私人联系方式和无关个人资料。
- 确认材料所有者、用途、保留与删除要求。

## 授权时

- 按任务授予最小资源、动作和时间范围。
- 区分读取、写入、发送、发布、删除；不要为读取授予全局写权限。
- 第三方 Skill 和连接器先审查来源、权限和数据去向。

## 执行与发布

- 事实、计算、文件和任务验收分层复核。
- 外部发送、发布、删除、覆盖和付费动作保留人工确认。
- 密钥只通过受控环境配置，不进入提示词、源码、日志或截图。

## 失败与事故

停止自动动作，保留脱敏记录，撤销临时权限并通知负责人。不要通过扩大权限、关闭检查或盲目重试掩盖根因。具体安全能力以当前官方版本和组织规则为准。
