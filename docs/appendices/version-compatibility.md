# 版本兼容

## 声明原则

蓝皮书不假定未实测的具体产品版本号。章节中的 `workbuddyVersion` 统一写“以当前官方版本为准”，并用 `lastVerified` 记录内容最近一次人工复核日期。

## 稳定与易变内容

任务合同、证据验证、最小权限等方法相对稳定；具体入口名称、支持文件、Skill、连接器、自动化和费用属于易变内容。执行前应查当前官方说明与组织策略。

## 发现差异时

记录章节、当前表现、官方依据、环境与日期。若只影响界面步骤，提交纠错；若改变权限或安全边界，应立即停止并升级审核。不要用旧截图推断新版本能力。
