# UI 与无障碍验收表

- 版本：1.0.0
- 关联章节：15、16
- 许可证：CC BY 4.0

## 路由与内容

- [ ] 路由、H1、页面目标与 PB-03 一致。
- [ ] 长标题、无内容、错误内容均有处理。

## 响应式

- [ ] 320px 宽无非预期横向滚动。
- [ ] 中屏导航和表格可用。
- [ ] 宽屏阅读行长合理。

## 无障碍

- [ ] 键盘可到达所有交互，焦点可见。
- [ ] 标题层级、按钮和表单标签语义正确。
- [ ] 对比度合格；颜色不是唯一信息载体。
- [ ] 图像有有效替代文本；图表有文字等价说明。
- [ ] 减少动效偏好得到尊重。

## SEO 与质量

- [ ] 唯一标题和描述、canonical、sitemap、robots。
- [ ] 生产构建、链接与内容校验通过。
- [ ] 发布内容、数据去向和回滚由人确认。
